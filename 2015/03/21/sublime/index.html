sublime text3 中安装JavaScript控制台

发表于 2015-03-21 | 分类于 开发工具 ， sublime

　　对于从事前端开发的小伙伴们来说，去控制台测试一下程序的输入输出是经常之事。多数情况下，我们会借助于 F12 这个功能。但对于一些小的 demo 和程序输出测试，使用 F12 难免会觉着有一些麻烦(对于 programmer 来说，一步能解决的事情，绝对不走第二步)。
　　webstorm这种 IDE 自不必说，atom 的 community packages 里面有 script 软件包，sublime 呢？
　　sublime 需要通过自身的 Build System 方法来让自身支持 JavaScript 控制台测试，安装步骤如下：
　　1. 安装nodeJs, win+r 打开cmd，中输入 node -v ，查看 node 是否被正确安装，如果出现版本号，则安装成功。
